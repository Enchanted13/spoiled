A free and open source spoiler-blocker
No ads. No tracking.

Spoiled covers up all text and images containing terms you provide.

Block Anything

Type any phrase to block it from websites you browse. You aren't limited by specific shows or presents. Spoiled was designed to make adding and removing phrases as fast and easy as possible.

Free and Open Source

I created Spoiled because I didn't trust that other blockers were handling my data with care. Many even ask for social network feed access. I thought making a quick little spoiler blocker would be a fun side project that maybe others would use as well. All the code is open source and we use no analytics of any kind so we have no access to the terms you enter or the sites you browse.
